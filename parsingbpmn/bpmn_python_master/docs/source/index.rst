Overview
--------

`bpmn_python` is Python client library for importing/exporting BPMN diagrams from an XML file with basic visualization capabilities using `matplotlib <https://matplotlib.org/>`_


Key features
------------

    - parse bpmn representation from xml files
    - tinker around and visualize bpmn models

Installation
------------

.. code-block:: bash

  pip install bpmn_python
